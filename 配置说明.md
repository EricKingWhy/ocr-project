# 百度OCR API配置说明

## 错误原因

当前出现的错误是因为 `config.py` 文件中的 API 密钥还未配置。程序需要百度OCR的API密钥才能正常工作。

## 配置步骤

### 1. 获取百度API密钥

1. 访问百度AI开放平台：https://ai.baidu.com/
2. 注册并登录账号（如果没有账号）
3. 进入控制台，创建新应用
4. 选择"文字识别"服务
5. 创建应用后，获取以下信息：
   - **API Key**（应用密钥）
   - **Secret Key**（应用密钥）

### 2. 配置到项目中

打开项目根目录下的 `config.py` 文件，将获取到的密钥填入：

```python
# config.py
API_KEY = "你的API_KEY"  # 替换为你的实际API Key
SECRET_KEY = "你的SECRET_KEY"  # 替换为你的实际Secret Key

TOKEN_URL = "https://aip.baidubce.com/oauth/2.0/token"
OCR_URL = "https://aip.baidubce.com/rest/2.0/ocr/v1/general_basic"
```

### 3. 保存并重启

保存 `config.py` 文件后，重新运行程序即可。

## 注意事项

- API密钥是敏感信息，请勿泄露给他人
- 百度OCR服务有免费额度，超出后需要付费
- 如果仍然出现错误，请检查：
  - API密钥是否正确
  - 网络连接是否正常
  - 百度AI平台中的应用是否已启用文字识别服务

## 测试配置

配置完成后，可以运行以下命令测试：

```bash
python auth.py
```

如果配置正确，会输出一个Token字符串；如果配置错误，会显示详细的错误信息。
